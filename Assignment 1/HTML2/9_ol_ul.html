Welcome to Order or Unorder List

  1. Strong Tag
    • The HTML strong element defines text with strong important.
    • The content inside typically display in bold
  2. I Tag
    • The HTML i element defines a part of text in an alternate voice or mood
    • The content inside is typically displayed in italic.
  3. Small Tag
    • The HTML small element defines smaller text
  4. Mark Tag
    • The HTML mark elemnt defines text that should be marked or highlighted.
