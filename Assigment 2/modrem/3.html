what is web accessibility?list some of the assistive devices which play a major role in providing accessibility

Web accessibility refers to the inclusive practice of designing and developing websites and web content in a way that can be accessed and used by individuals with disabilities.
It aims to ensure that people with disabilities can perceive, understand, navigate, and interact with websites effectively.

Assistive devices are tools or technologies that assist individuals with disabilities in accessing and using web content.
Here are some common assistive devices that play a major role in providing web accessibility:

  1. Screen Readers
  2. Screen Magnifiers
  3. Alternative Input Devices
  4. Text-to-Speech Tools
